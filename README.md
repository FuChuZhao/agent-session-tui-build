# agent-session-tui-build

This repository hosts GitHub Actions workflows to build `agent-session-tui` from the private source repo `FuChuZhao/agent-session-tui`.

## Usage

- Go to **Actions** → **build** → **Run workflow**
- Set `ref` (branch/tag/sha) from the source repo (default: `main`)
- Download the artifact `agent-session-tui-windows-x86_64.zip`
